cc3的出现

我们发现上面的cc1和cc6都有一个共同的特点就是都用到了InvokerTransformer；如今在很多防护中早已经把InvokerTransformer放进黑名单，这时我们不得不寻找一个新的利用链进行绕过。

我们知道在TemplatesImpl执行命令中是调用newTransformer来实例化的，我们再看看还有没有其它地方调用它的。
